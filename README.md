# E-Commerce-App

A clone application that hosts the classic transactions that an e-commerce site can do.
<hr>
Features:<br>
-Login/Register<br>
-Login/Register with Google<br>
-Reset Password<br>
-List products/categories<br>
-Product Details
-Add to Cart<br>
-Remove from Cart<br>
-Create Order<br>
-Admin Panel<br>
